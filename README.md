# 🎮 Video Game Sales Analytics Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** built using a comprehensive video game sales dataset. The dashboard analyzes trends across game genres, platforms, publishers, global sales, critic scores, user ratings, and yearly releases to uncover meaningful industry insights.

The goal of this project is to transform raw sales data into actionable business intelligence using data modeling, DAX calculations, and visual storytelling.

---

## 📊 Dashboard Features

* Genre-wise comparison of global sales and number of games
* Platform contribution to worldwide revenue
* Publisher-level performance analysis using critic and user ratings
* Rating-wise distribution of video games
* Yearly trend of game releases
* Top-performing games based on global sales

---

## 🔍 Key Insights

* Action and Sports genres dominate both sales and volume.
* Platforms such as **X360, PS3, and PS2** contribute significantly to global revenue.
* Game releases peaked during the mid-2000s.
* Publisher performance varies widely based on critic and user ratings.
* Certain titles stand out as global best-sellers.
